# AI DevOps Self-Healing Agent

An AI-powered system that monitors Kubernetes workloads, detects anomalies, and automatically analyzes and resolves issues using LLM-based reasoning.

## Key Features
- Real-time monitoring of containerized applications
- AI-driven anomaly detection
- Automated incident analysis using LLMs
- Self-healing infrastructure actions

## Tech Stack
- Python, FastAPI
- Kubernetes (Minikube)
- Docker
- Prometheus, Grafana
- OpenAI API

## Overview
This project simulates an intelligent DevOps agent that reduces manual intervention by detecting and resolving infrastructure issues automatically. It is designed with a production-ready architecture and can scale to real-world environments, especially when integrated with full LLM capabilities via paid API access.
